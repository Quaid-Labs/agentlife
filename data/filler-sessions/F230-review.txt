MAYA:
  wait can you explain async vs await real quick? i keep mixing them up lol

AI ASSISTANT:
  async marks a function as asynchronous (lets it use await inside). await pauses execution until a promise resolves. So async = function declaration, await = pausing for results inside that function.

AI ASSISTANT:
  Exactly! async function = "this function can pause and wait for things". await = "pause here until this promise finishes". You need async to use await.
