# Published Benchmark Artifacts

This tree is the tracked home for release-ready public benchmark materials.

Use it for:

- cleaned public markdown summaries
- release-backed runbooks
- frozen benchmark checkpoint artifacts that support published claims

Do not use it for:

- live scratch runs
- temporary forensics
- internal WIP matrices
- recovered remote snapshots

Layout:

- `runbooks/`
  - release-ready benchmark summaries and comparison matrices
- `checkpoints/`
  - frozen public-supporting artifacts grouped by release tag

For active release prep, use the tracked `release-candidate/` subdirectories
first, then rename or copy them to a real release tag once the numbers and
artifacts are approved.
