# Selected Eval Results

This directory is intentionally still sparse during overnight prep.

Use it for a small, reviewed subset of eval artifacts that directly support
public release claims, for example:

- representative `evaluation_results.json` excerpts for a flagship lane
- curated miss examples that explain a benchmark tradeoff
- short supporting artifacts used in a press or release writeup

Do not dump full scratch rerun trees here. Keep this directory curated and
claim-driven.

Current status:

- score and usage artifacts are already staged in sibling directories
- detailed eval excerpts are still pending manual curation during release review
